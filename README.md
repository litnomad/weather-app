# Weather app

A weather forecast site

## Objective

- The objective of this project is to create a weather app using asynchronous code—promises or async/await—and API fetching.

## Features

- location based
- appearance updates according to data

See [example](https://htmlpreview.github.io/?https://github.com/litnomad/weather-app/blob/38c4c018dba9e03bfff09503d225e2936a0761d3/index.html).
